# Architecture

## System Overview

## Key Design Decisions

1. **SVG-based rendering** вҖ” Chosen over Canvas for easier DOM interaction (click/drag on individual elements)
2. **A* with half-step grid** вҖ” Wires snap to 0.5-unit increments for diagonal routing
3. **Dual persistence** вҖ” localStorage for offline, Firestore for cloud sync (debounced 2s)
4. **Monolithic `App.tsx`** вҖ” Generated by AI Studio; structural refactoring planned for future iteration
